# Android 手机部署 SillyTavern 教程（Termux 一键脚本）

> 作者：10091009mc  
> 脚本名称：Angler Toolbox（`angler_toolbox.sh`）  
> 适用场景：**仅 Android + Termux**，不需要电脑  
> 许可说明：脚本免费，仅供学习与个人使用，禁止商业倒卖

---

## 这篇教程写给谁

如果你是下面这类用户，这份教程就是给你准备的：

- 想在手机上部署 SillyTavern，但不想折腾复杂命令
- 需要一个“安装、更新、备份、保活”都能覆盖的方案
- 希望出问题后能快速恢复，而不是重装一切

---

## 教程目标（你将获得什么）

按本教程完成后，你可以做到：

1. 在 Android 手机上安装并启动 SillyTavern
2. 用菜单完成更新、回退、备份与恢复
3. 开启保活（降低被系统杀后台的概率）
4. 可选配置开机自启（按需开启）

---

## 0. 准备工作（必须做）

1. 从 [F-Droid](https://f-droid.org/zh_Hant/packages/com.termux/) 或者https://github.com/termux/termux-app
2. 打开 Termux，复制并执行：

   ```bash
   bash <(curl -s https://raw.githubusercontent.com/mc10091009/st_manager.sh/main/angler_toolbox.sh)
   ```

脚本会自动完成依赖检查与安装，包括：

`curl`、`git`、`nodejs`、`python`、`build-essential`、`tar`、`jq`、`lsof`、`psmisc`、`procps`、`termux-api`

## 1. 第一次运行：你会看到什么

首次运行通常会自动完成以下流程：

1. 检查并补齐依赖
2. 下载/安装 SillyTavern
3. 进入功能菜单
4. 可选提示你是否配置开机自启

如果中途网络波动导致失败，重新执行一次脚本即可。

## 3. 推荐使用流程（新手照做即可）

### 安装并启动

1. 菜单输入 `2` 挑選版本1.13.4/1.14.0/或者1.16.0 不過當中的1.16.0渲染壓力很大，然後輸入版本號安装 SillyTavern  
2. 完成后输入 `1` 启动  
3. 按提示访问本机地址（通常是 `127.0.0.1:8000` 或脚本显示的地址）


### 需要更新时

1. 菜单输入 `3`
2. 选择跟踪 `release` 或 `main`，也可指定 Tag
3. 更新后若异常，去菜单 `4` 回退稳定版本

##  开机自启（可选）

- 菜单 `10` 可开启/关闭开机自启
- 脚本会自动处理 shell 配置，清理旧残留
- 如果你不需要自启，保持关闭即可

## 4. 常见问题（教程版）

### Q1：`npm install` 失败怎么办？

先检查网络，再执行菜单 `5`（重装依赖）。  
大多数依赖损坏问题可通过该菜单修复。

### Q2：提示端口被占用但我没开别的程序？

执行菜单 `7` 做端口检查与清理，按提示结束残留进程。

### Q3：更新后不稳定怎么办？

使用菜单 `4` 回退到之前可用的 Tag/Commit。  
**更新前先备份**，恢复会更快。

### Q4：担心误删数据？

卸载相关功能有多重确认和验证码，防止误操作。  
但依然建议先在菜单 `6` 手动备份。
